# The Friday Forge

A 36-week Friday tech masterclass by [Tequity](https://tequity.tech).

**30-60 min every Friday. Live coding. No slides. All levels welcome.**

---

## Series

| # | Weeks | Topic | What You'll Learn |
|---|-------|-------|-------------------|
| 01 | 1–12 | **Design Patterns** | Write cleaner, more maintainable code using proven solutions |
| 02 | 13–24 | **System Design** | Think in systems — APIs, caching, queues, scaling |
| 03 | 25–36 | **DevOps** | Ship code from your laptop to production with confidence |

## How to Use This Repo

Each week's folder contains runnable code examples from the session. Clone and experiment:

```bash
git clone https://github.com/shubham-tequity/the-friday-forge.git
cd the-friday-forge/week-01-why-design-patterns-matter
node 01-the-problem.js
node 02-the-fix.js
```
